C Language Notes: First Year Fundamentals

Welcome to C Programming!

C is a powerful, general-purpose programming language developed by Dennis Ritchie at Bell Labs in the early 1970s. It's known for its efficiency, flexibility, and close-to-hardware capabilities, making it a foundational language for operating systems, embedded systems, and high-performance applications. Understanding C is crucial for anyone looking to delve deep into computer science.

These notes are designed to be straightforward and easy to understand for first-year students, with clear explanations and practical code examples.

1. Getting Started: Hello World!

1.1 Basic Structure of a C Program

A typical C program starts with the `main` function, which is the entry point of execution. The `printf` function is used to display output to the console. 

#include <stdio.h>

int main() {
    printf("Hello, BOZON ACADEMY!\n");
    return 0;
}

Explanation:
#include <stdio.h>: This line includes the standard input/output library, which provides functions like `printf`.
int main(): This is the main function where program execution begins. `int` means it returns an integer value.
printf("Hello, BOZON ACADEMY!\n");: This prints the string "Hello, BOZON ACADEMY!" to the console. `\n` creates a new line.
return 0;: This indicates that the program executed successfully.

1.2 Compiling and Running C Programs

C is a compiled language. This means you write your code, then use a compiler (like GCC) to translate it into an executable file that your computer can run. 

# Save your code as hello.c
# Compile:
gcc hello.c -o hello
# Run:
./hello

2. Variables and Data Types

Variables are named storage locations that hold data. In C, you must declare a variable's data type before using it. This tells the compiler how much memory to allocate and what kind of values the variable can store.

2.1 Basic Data Types

  • int: for integers (whole numbers like -5, 0, 100). Typically 2 or 4 bytes.
  • float: for single-precision floating-point numbers (decimal numbers like 3.14, -0.5). Typically 4 bytes.
  • double: for double-precision floating-point numbers (more precise decimal numbers). Typically 8 bytes.
  • char: for single characters (like 'A', 'b', '7'). Typically 1 byte.
  • void: Represents the absence of type. Used for functions that return no value or generic pointers.
#include <stdio.h>

int main() {
    int age = 20;
    float price = 19.99;
    char grade = 'A';
    double pi = 3.1415926535;

    printf("Age: %d\n", age);
    printf("Price: %.2f\n", price);
    printf("Grade: %c\n", grade);
    printf("Pi: %lf\n", pi);
    return 0;
}

Explanation of Format Specifiers:
%d: Used for printing `int` values.
%.2f: Used for printing `float` or `double` values, showing two decimal places.
%c: Used for printing `char` values.
%lf: Used for printing `double` values (note: `%f` also works for `double` in `printf`, but `%lf` is required for `scanf`).

2.2 Type Modifiers

Modifiers can be used with basic data types to change their range or behavior:

  • short: Smaller range than `int`.
  • long: Larger range than `int`.
  • signed: Can hold positive and negative values (default).
  • unsigned: Can only hold non-negative values, extending the positive range.

#include <stdio.h>

int main() {
    unsigned int positive_num = 4000000000; // Larger positive value
    long int big_num = 1234567890123L;     // Very large integer
    printf("Unsigned Int: %u\n", positive_num);
    printf("Long Int: %ld\n", big_num);
    return 0;
}

3. Operators

Operators are symbols that tell the compiler to perform specific mathematical, relational, or logical operations and produce a final result.

3.1 Arithmetic Operators

  • `+` (Addition)
  • `-` (Subtraction)
  • `*` (Multiplication)
  • `/` (Division)
  • `%` (Modulus - remainder of division)
  • `++` (Increment - adds 1)
  • `--` (Decrement - subtracts 1)
#include <stdio.h>

int main() {
    int x = 10, y = 3;
    printf("x + y = %d\n", x + y); // 13
    printf("x / y = %d\n", x / y); // 3 (integer division)
    printf("x %% y = %d\n", x % y); // 1
    x++; // x becomes 11
    printf("x after increment: %d\n", x);
    return 0;
}

3.2 Relational Operators

Used to compare two values. They return 1 (true) or 0 (false).

  • `==` (Equal to)
  • `!=` (Not equal to)
  • `>` (Greater than)
  • `<` (Less than)
  • `>=` (Greater than or equal to)
  • `<=` (Less than or equal to)
#include <stdio.h>

int main() {
    int a = 5, b = 5, c = 10;
    printf("a == b: %d\n", a == b); // 1 (True)
    printf("a != c: %d\n", a != c); // 1 (True)
    printf("a > c: %d\n", a > c);   // 0 (False)
    return 0;
}

3.3 Logical Operators

Used to combine or negate conditional statements.

  • `&&` (Logical AND - true if both operands are true)
  • `||` (Logical OR - true if at least one operand is true)
  • `!` (Logical NOT - reverses the logical state)
#include <stdio.h>

int main() {
    int age = 25;
    int has_license = 1; // 1 for true, 0 for false

    if (age >= 18 && has_license) {
        printf("Eligible to drive.\n");
    }
    if (age < 18 || !has_license) {
        printf("Not eligible to drive.\n");
    }
    return 0;
}

3.4 Assignment Operators

Used to assign values to variables.

  • `=` (Simple assignment)
  • `+=` (Add and assign: `x += y` is same as `x = x + y`)
  • `-=` (Subtract and assign)
  • `*=` (Multiply and assign)
  • `/=` (Divide and assign)
  • `%=` (Modulus and assign)
#include <stdio.h>

int main() {
    int num = 10;
    num += 5; // num is now 15
    printf("num: %d\n", num);
    num *= 2; // num is now 30
    printf("num: %d\n", num);
    return 0;
}

4. Control Flow Statements

Control flow statements dictate the order in which instructions are executed in a program.

4.1 Conditionals: `if`, `else if`, `else`

Allow your program to make decisions based on certain conditions. 

#include <stdio.h>

int main() {
    int score = 75;
    if (score >= 90) {
        printf("Grade: A\n");
    } else if (score >= 80) {
        printf("Grade: B\n");
    } else if (score >= 70) {
        printf("Grade: C\n");
    } else {
        printf("Grade: F\n");
    }
    return 0;
}

4.2 Conditionals: `switch` Statement

A multi-way branch statement that provides an easy way to dispatch execution to different parts of code based on the value of an expression. 

#include <stdio.h>

int main() {
    char grade = 'B';
    switch (grade) {
        case 'A':
            printf("Excellent!\n");
            break;
        case 'B':
            printf("Very Good!\n");
            break;
        case 'C':
            printf("Good.\n");
            break;
        default:
            printf("Needs Improvement.\n");
    }
    return 0;
}

4.3 Loops: `for` Loop

Used when you know exactly how many times you want to loop. 

#include <stdio.h>

int main() {
    printf("Numbers from 1 to 5:\n");
    for (int i = 1; i <= 5; i++) {
        printf("%d ", i);
    }
    printf("\n");
    return 0;
}

4.4 Loops: `while` Loop

Used when you want to loop as long as a condition is true. 

#include <stdio.h>

int main() {
    printf("Counting down from 5:\n");
    int count = 5;
    while (count > 0) {
        printf("%d ", count);
        count--;
    }
    printf("\n");
    return 0;
}

4.5 Loops: `do-while` Loop

Similar to `while`, but guarantees the loop body executes at least once. 

#include <stdio.h>

int main() {
    int i = 0;
    do {
        printf("Hello (do-while)!\n");
        i++;
    } while (i < 1); // Condition is false, but runs once
    return 0;
}

4.6 `break` and `continue` Statements

`break` exits the loop immediately. `continue` skips the current iteration and proceeds to the next. 

#include <stdio.h>

int main() {
    printf("Using break:\n");
    for (int i = 1; i <= 5; i++) {
        if (i == 3) {
            break; // Exit loop when i is 3
        }
        printf("%d ", i);
    }
    printf("\n");

    printf("Using continue:\n");
    for (int i = 1; i <= 5; i++) {
        if (i == 3) {
            continue; // Skip 3
        }
        printf("%d ", i);
    }
    printf("\n");
    return 0;
}

5. Functions

Functions are blocks of code that perform a specific task. They promote modularity, reusability, and make programs easier to understand and debug.

5.1 Function Declaration (Prototype) and Definition

A function prototype tells the compiler about a function's name, return type, and parameters before its actual definition. 

#include <stdio.h>

// Function declaration (prototype)
int add(int num1, int num2);

int main() {
    int result = add(10, 5);
    printf("Sum: %d\n", result);
    return 0;
}

// Function definition
int add(int num1, int num2) {
    return num1 + num2;
}

5.2 Parameters and Return Values

Functions can accept input values (parameters) and can return a single value back to the calling code. 

#include <stdio.h>

// Function that takes no parameters and returns no value
void greet() {
    printf("Hello there!\n");
}

// Function that takes a parameter and returns no value
void print_number(int num) {
    printf("The number is: %d\n", num);
}

// Function that takes parameters and returns a value
int multiply(int a, int b) {
    return a * b;
}

int main() {
    greet();
    print_number(42);
    int product = multiply(7, 6);
    printf("Product: %d\n", product);
    return 0;
}

6. Arrays

An array is a collection of elements of the same data type stored at contiguous memory locations. They are useful for storing a fixed-size sequential collection of elements.

6.1 Declaring and Initializing Arrays

#include <stdio.h>

int main() {
    // Declare an array of 5 integers
    int numbers[5];

    // Initialize elements
    numbers[0] = 10;
    numbers[1] = 20;
    numbers[2] = 30;
    numbers[3] = 40;
    numbers[4] = 50;

    // Declare and initialize an array at once
    float prices[] = {1.99, 2.49, 0.75}; // Size is automatically determined (3)

    printf("First number: %d\n", numbers[0]);
    printf("Second price: %.2f\n", prices[1]);
    return 0;
}

6.2 Accessing Array Elements

Array elements are accessed using an index, which starts from 0 for the first element. 

#include <stdio.h>

int main() {
    int scores[4] = {85, 90, 78, 92};

    printf("Scores: ");
    for (int i = 0; i < 4; i++) {
        printf("%d ", scores[i]);
    }
    printf("\n");
    return 0;
}

6.3 Multi-dimensional Arrays (Basic)

Arrays can have more than one dimension, like a grid or a table. A 2D array is often called a matrix. 

#include <stdio.h>

int main() {
    int matrix[2][3] = {
        {1, 2, 3},
        {4, 5, 6}
    };

    printf("Element at [0][1]: %d\n", matrix[0][1]); // Output: 2
    printf("Element at [1][2]: %d\n", matrix[1][2]); // Output: 6
    return 0;
}

7. Strings

In C, strings are essentially arrays of characters terminated by a null character (`\0`). The `` library provides many useful functions for string manipulation.

7.1 Declaring and Initializing Strings

#include <stdio.h>
#include <string.h> // For string functions

int main() {
    char greeting[] = "Hello"; // Array of characters, size determined automatically
    char name[20]; // Array to hold up to 19 characters + null terminator

    printf("Greeting: %s\n", greeting);

    // Copying a string
    strcpy(name, "Bozon");
    printf("Name: %s\n", name);

    // Concatenating strings
    strcat(greeting, " World"); // greeting is now "Hello World"
    printf("Concatenated: %s\n", greeting);

    // String length
    printf("Length of greeting: %lu\n", strlen(greeting)); // %lu for size_t
    return 0;
}

8. Pointers

Pointers are variables that store memory addresses. They are a fundamental and powerful concept in C, allowing for direct memory manipulation, efficient array handling, and dynamic memory allocation.

8.1 Declaring and Initializing Pointers

  • `*` (Dereference operator): Used to access the value at the address pointed to by the pointer.
  • `&` (Address-of operator): Used to get the memory address of a variable.
#include <stdio.h>

int main() {
    int var = 10;
    int *ptr; // Declare a pointer variable that points to an integer

    ptr = &var; // Store the address of 'var' in 'ptr'

    printf("Value of var: %d\n", var);
    printf("Address of var: %p\n", &var); // %p for printing address
    printf("Value of ptr (address it holds): %p\n", ptr);
    printf("Value at the address ptr holds (*ptr): %d\n", *ptr); // Dereferencing

    *ptr = 20; // Change value of var using pointer
    printf("New value of var: %d\n", var);
    return 0;
}

8.2 Pointers and Arrays

Array names can often be treated as pointers to their first element. This close relationship is key to C's power. 

#include <stdio.h>

int main() {
    int arr[] = {10, 20, 30, 40, 50};
    int *p = arr; // p points to the first element (arr[0])

    printf("arr[0] using pointer: %d\n", *p);
    printf("arr[1] using pointer arithmetic: %d\n", *(p + 1)); // Moves to next element
    printf("arr[2] using pointer arithmetic: %d\n", *(arr + 2)); // arr itself can be used

    return 0;
}

9. Structures, Unions, and Enums

These are user-defined data types that allow you to group related data and define custom sets of named integer constants.

9.1 Structures (`struct`)

A `struct` is a collection of variables (of different data types) under a single name. It's used to create complex data types. 

#include <stdio.h>
#include <string.h>

// Define a structure for a student
struct Student {
    int id;
    char name[50];
    float grade;
};

int main() {
    struct Student s1; // Declare a structure variable

    s1.id = 101;
    strcpy(s1.name, "Alice Smith");
    s1.grade = 88.5;

    printf("Student ID: %d\n", s1.id);
    printf("Student Name: %s\n", s1.name);
    printf("Student Grade: %.1f\n", s1.grade);
    return 0;
}

9.2 Unions (`union`)

Similar to structures, but all members share the same memory location. Only one member can hold a value at any given time. Useful for memory optimization when you know only one piece of data will be active. 

#include <stdio.h>
#include <string.h>

union Data {
    int i;
    float f;
    char str[20];
};

int main() {
    union Data data;

    data.i = 10;
    printf("data.i: %d\n", data.i);

    data.f = 220.5; // This overwrites data.i
    printf("data.f: %.1f\n", data.f);

    strcpy(data.str, "C Programming"); // This overwrites data.f
    printf("data.str: %s\n", data.str);

    // Accessing other members after str is set will give garbage values
    printf("data.i (after str): %d\n", data.i);
    return 0;
}

9.3 Enumerations (`enum`)

An `enum` is a user-defined data type that consists of a set of named integer constants. It makes code more readable and maintainable. 

#include <stdio.h>

enum Weekday {
    MONDAY,    // 0 by default
    TUESDAY,   // 1
    WEDNESDAY, // 2
    THURSDAY,  // 3
    FRIDAY,    // 4
    SATURDAY,  // 5
    SUNDAY     // 6
};

int main() {
    enum Weekday today = WEDNESDAY;
    printf("Today is day number: %d\n", today); // Output: 2

    if (today == WEDNESDAY) {
        printf("It's hump day!\n");
    }
    return 0;
}

10. File I/O (Input/Output)

File I/O allows your C programs to interact with files on your computer's disk, enabling persistence of data.

10.1 Opening and Closing Files

The `FILE` pointer is used to manage files. `fopen()` opens a file, and `fclose()` closes it.

  • `"r"`: Read mode (file must exist)
  • `"w"`: Write mode (creates new file or overwrites existing)
  • `"a"`: Append mode (creates new file or adds to end of existing)
  • `"r+"`: Read and write (file must exist)
  • `"w+"`: Read and write (creates new file or overwrites existing)
  • `"a+"`: Read and append (creates new file or adds to end of existing)
#include <stdio.h>

int main() {
    FILE *fptr;

    // Open file in write mode
    fptr = fopen("example.txt", "w");

    if (fptr == NULL) {
        printf("Error opening file!\n");
        return 1;
    }

    printf("File opened successfully for writing.\n");
    fclose(fptr); // Close the file
    printf("File closed.\n");
    return 0;
}

10.2 Writing to Files

Use `fprintf()` to write formatted output to a file, similar to `printf()`. 

#include <stdio.h>

int main() {
    FILE *fptr;
    fptr = fopen("data.txt", "w");

    if (fptr == NULL) {
        printf("Error opening file!\n");
        return 1;
    }

    fprintf(fptr, "Hello from C programming!\n");
    fprintf(fptr, "This is a new line.\n");
    fclose(fptr);
    printf("Data written to data.txt\n");
    return 0;
}

10.3 Reading from Files

Use `fscanf()` to read formatted input from a file, similar to `scanf()`. `fgets()` can read a whole line. 

#include <stdio.h>

int main() {
    FILE *fptr;
    char buffer[100];

    fptr = fopen("data.txt", "r"); // Open the file we just wrote to

    if (fptr == NULL) {
        printf("Error opening file!\n");
        return 1;
    }

    printf("Content of data.txt:\n");
    while (fgets(buffer, sizeof(buffer), fptr) != NULL) {
        printf("%s", buffer);
    }

    fclose(fptr);
    return 0;
}

11. Dynamic Memory Allocation

Dynamic memory allocation allows you to allocate memory during program execution (runtime), rather than at compile time. This is essential when you don't know the exact memory requirements beforehand. The `` header provides functions for this.

11.1 `malloc()` (Memory Allocation)

Allocates a block of memory of the specified size and returns a `void` pointer to the beginning of the block. You must cast it to the desired type. 

#include <stdio.h>
#include <stdlib.h> // For malloc, free

int main() {
    int *ptr;
    int n = 5;

    // Allocate memory for 5 integers
    ptr = (int*) malloc(n * sizeof(int));

    if (ptr == NULL) {
        printf("Memory allocation failed!\n");
        return 1;
    }

    printf("Memory allocated successfully.\n");

    // Initialize and print allocated memory
    for (int i = 0; i < n; i++) {
        ptr[i] = i + 1;
        printf("%d ", ptr[i]);
    }
    printf("\n");

    free(ptr); // Free the allocated memory
    printf("Memory freed.\n");
    return 0;
}

11.2 `calloc()` (Contiguous Allocation)

Allocates memory for a specified number of elements of a given size and initializes all bytes to zero. 

#include <stdio.h>
#include <stdlib.h>

int main() {
    int *ptr;
    int n = 3;

    // Allocate memory for 3 integers and initialize to 0
    ptr = (int*) calloc(n, sizeof(int));

    if (ptr == NULL) {
        printf("Memory allocation failed!\n");
        return 1;
    }

    printf("Calloc allocated memory (initialized to 0): ");
    for (int i = 0; i < n; i++) {
        printf("%d ", ptr[i]);
    }
    printf("\n");

    free(ptr);
    return 0;
}

11.3 `realloc()` (Reallocate Memory)

Changes the size of the memory block pointed to by `ptr` to the new specified size. 

#include <stdio.h>
#include <stdlib.h>

int main() {
    int *ptr;
    ptr = (int*) malloc(2 * sizeof(int)); // Allocate for 2 ints
    ptr[0] = 10;
    ptr[1] = 20;

    printf("Original: %d %d\n", ptr[0], ptr[1]);

    // Reallocate to hold 4 integers
    ptr = (int*) realloc(ptr, 4 * sizeof(int));
    if (ptr == NULL) {
        printf("Reallocation failed!\n");
        return 1;
    }

    ptr[2] = 30;
    ptr[3] = 40;
    printf("Reallocated: %d %d %d %d\n", ptr[0], ptr[1], ptr[2], ptr[3]);

    free(ptr);
    return 0;
}

11.4 `free()` (Free Memory)

Deallocates the memory previously allocated by `malloc()`, `calloc()`, or `realloc()`. It's crucial to `free` memory to prevent memory leaks. 

#include <stdio.h>
#include <stdlib.h>

int main() {
    int *ptr = (int*) malloc(sizeof(int));
    if (ptr == NULL) return 1;
    *ptr = 100;
    printf("Value: %d\n", *ptr);
    free(ptr); // Memory is returned to the system
    // ptr is now a dangling pointer, do not use it again without reassigning
    // ptr = NULL; // Good practice to set to NULL after freeing
    return 0;
}

12. Preprocessor Directives

The C preprocessor is a program that processes the source code before compilation. Directives begin with a `#` symbol.

12.1 `#include`

Includes the content of another file into the current source file.

  • `#include `: For standard library headers.
  • `#include "filename"`: For user-defined headers.

#include <stdio.h> // Standard library
#include "my_header.h" // User-defined header (if it exists)

12.2 `#define`

Used to define macros, which are essentially text substitutions. 

#include <stdio.h>

#define PI 3.14159
#define MAX(a, b) ((a > b) ? a : b)

int main() {
    printf("Value of PI: %.5f\n", PI);
    printf("Max of 10 and 20: %d\n", MAX(10, 20));
    return 0;
}

12.3 Conditional Compilation (`#ifdef`, `#ifndef`, `#if`, `#else`, `#endif`)

Allows you to include or exclude parts of code based on conditions. 

#include <stdio.h>

#define DEBUG_MODE

int main() {
    #ifdef DEBUG_MODE
        printf("Debugging is enabled.\n");
    #else
        printf("Debugging is disabled.\n");
    #endif

    #if __STDC_VERSION__ >= 199901L
        printf("C99 or later standard.\n");
    #else
        printf("Older C standard.\n");
    #endif
    return 0;
}

13. Command Line Arguments

C programs can accept arguments directly from the command line when they are executed. These are passed to the `main` function.

  • `argc`: (Argument Count) An integer representing the number of arguments passed.
  • `argv`: (Argument Vector) An array of strings (char pointers), where each string is one of the arguments. `argv[0]` is always the program's name.
#include <stdio.h>

int main(int argc, char *argv[]) {
    printf("Number of arguments: %d\n", argc);
    printf("Arguments received:\n");
    for (int i = 0; i < argc; i++) {
        printf("Argument %d: %s\n", i, argv[i]);
    }
    return 0;
}

How to run:
Compile: `gcc your_program.c -o your_program`
Run: `./your_program arg1 arg2 "another argument"`
